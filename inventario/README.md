## Programa de Inventario
### Descripción

Este proyecto es un sencillo sistema de inventario escrito en Python. Se trata de una aplicación de consola que permite al usuario registrar productos, incluyendo su nombre, precio y cantidad.

El programa está diseñado para programadores principiantes que están aprendiendo Python. Muestra cómo trabajar con bucles, entrada de datos por parte del usuario y validación de datos.

### Funciones

- Registrar productos en el inventario
- Validar el nombre del producto
- Validar el precio
- Validar la cantidad
- Generar una factura simple
- Calcular el costo total de los productos

### Funcionamiento

1. El programa muestra un mensaje de bienvenida.
2. El usuario ingresa el nombre de un producto.
3. El programa verifica que el nombre contenga solo letras y espacios.
4. El usuario ingresa el precio unitario del producto.
5. El programa valida que el precio sea un número positivo.
6. El usuario ingresa la cantidad del producto.
7. El programa valida que la cantidad sea un número entero positivo.
8. El programa calcula el costo total.
9. El programa imprime una factura simple con la información del producto.
10. El programa pregunta al usuario si desea registrar otro producto.
